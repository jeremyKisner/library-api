# Library API
Start the API server:
``` bash
python3 .\library-api\main\app.py
```

To run tests for the API:
``` bash
python3 -m pytest .\library-api\tests\
```


# Library UI
Now, change directory into the library-ui:
``` bash
cd library-ui
```

Then, start the UI:
``` bash
npm start
```
